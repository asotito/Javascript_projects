# ONLY JAVASCRIPT PROJECTS

## Training projects recommended by ChatGPT

### Español
#### 1. Generador de contraseñas personalizables
- Aplicación donde el usuario selecciona longitud, tipos de caracteres (mayúsculas, minúsculas, números, símbolos) y genera una contraseña aleatoria. Practicas lógica, manejo de eventos y manipulación del DOM.

#### 2. Conversor de divisas con tasas obtenidas de una API
- Ingresas un monto, eliges dos monedas y obtienes el valor convertido usando una API pública. Practicas asincronía, fetch, manejo de errores y actualización dinámica de la interfaz.

#### 3. Administrador de tareas con filtros y persistencia
- Lista donde puedes agregar, editar, eliminar y marcar tareas como completadas. Incluye filtros (todas, activas, completadas) y guarda todo en localStorage. Practicas CRUD, estructuras de datos y renderizado condicional.

#### 4. Simulador de carrito de compras
- Catálogo simple donde puedes agregar productos, modificar cantidades, eliminar ítems y ver el total actualizado. Practicas arrays, objetos, eventos, cálculos dinámicos y almacenamiento local.

#### 5. Juego de memoria (cartas ocultas)
- Cartas volteadas donde debes encontrar pares iguales. Incluye temporizador, sistema de puntuación y niveles de dificultad. Practicas algoritmos, temporizadores, DOM y estado del juego.

#### 6. Dashboard de clima
- Buscas una ciudad y obtienes información actual (temperatura, viento, humedad) desde una API. Puede incluir búsqueda, historial y recarga automática. Practicas asincronía, consumo de APIs y control de estados.

#### 7. Reproductor de música
- Interfaz con play, pause, siguiente, anterior, barra de progreso y volumen. Lista de reproducción dinámica y control del estado del audio. Practicas eventos, objetos multimedia, estados y manejo del DOM.

#### 8. Sistema de reservas (citas o turnos)
- Agenda donde eliges fecha, hora y recurso disponible. Maneja validaciones (horarios ocupados, indisponibles) y estados de reserva. Practicas estructuras de datos, validaciones, lógica compleja y persistencia.

#### 9. Editor de texto simple
- Área de edición con botones para negrita, cursiva, tamaño, color y guardado del contenido. También cargas versiones anteriores. Practicas manipulación del DOM, eventos y manejo de estados.

#### 10. Chat offline con usuarios ficticios
- Chat donde simulas mensajes enviados/recibidos, tiempos, estados “escribiendo…”, notificaciones y distintos usuarios. Practicas asincronía simulada, temporizadores, eventos y estado global de la aplicación.

### English

#### 1. Customizable password generator

* An app where the user selects length, character types (uppercase, lowercase, numbers, symbols) and generates a random password. You practice logic, event handling, and DOM manipulation.

#### 2. Currency converter using API rates

* You enter an amount, choose two currencies, and get the converted value using a public API. You practice asynchrony, fetch, error handling, and dynamic UI updates.

#### 3. Task manager with filters and persistence

* A list where you can add, edit, delete, and mark tasks as completed. Includes filters (all, active, completed) and saves everything to localStorage. You practice CRUD, data structures, and conditional rendering.

#### 4. Shopping cart simulator

* A simple catalog where you add products, change quantities, remove items, and view the updated total. You practice arrays, objects, events, dynamic calculations, and local storage.

#### 5. Memory game (matching cards)

* Cards are placed face-down and you must find matching pairs. Includes a timer, scoring system, and difficulty levels. You practice algorithms, timers, DOM manipulation, and game state logic.

#### 6. Weather dashboard

* Search for a city and get current data (temperature, wind, humidity) from an API. May include search history and auto-refresh. You practice asynchrony, API consumption, and state management.

#### 7. Music player

* Interface with play, pause, next, previous, progress bar, and volume control. Includes a dynamic playlist and audio state handling. You practice events, multimedia objects, state management, and DOM updates.

#### 8. Booking system (appointments or time slots)

* A scheduler where users select a date, time, and an available resource. Handles validations for occupied or unavailable slots. You practice data structures, validations, complex logic, and persistence.

#### 9. Simple text editor

* Editing area with buttons for bold, italic, font size, color, and saving content. Also supports loading previous versions. You practice DOM manipulation, events, and state handling.

#### 10. Offline chat simulation with fake users

* A mock chat where you simulate sent/received messages, typing indicators, timestamps, and multiple users. You practice simulated asynchrony, timers, event handling, and global state management.
